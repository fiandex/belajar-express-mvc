# Belajar Express JS - MVC Architecture

Ini adalah project latihan Backend menggunakan:
- Express.js
- PostgreSQL (Neon Tech)
- Prisma ORM
- Zod Validation
- Service Layer Architecture

## Cara Menjalankan
1. Clone repo ini
2. `npm install`
3. Setup `.env` (DATABASE_URL)
4. `npx prisma db push`
5. `npm run dev`
